Greenfoot Programming project
TCC School Simulator

By: Ethan, Chris, and Andrew.

This is a simulator of the Virginia Beach TCC campus. To go to certain buildings, just click on the designated building such as the student
center. Doing so would take you to the exterior of the building. Clicking on other keys would take you to the rooms associated with that
building.

The image resources contains many original images from accross TCC Campus courtesy of Chris.

Greenfoot Version 2.4.2

Java Version 1.8.0

Developed on Microsoft Windows 7-8.1, 10.
